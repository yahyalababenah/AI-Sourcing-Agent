```bash

# 2. ماذا أنجزنا؟
```

  أكمله واعمل commit. إن كان مكسوراً: `git checkout .` وأعد المهمة.

## بروتوكول نهاية كل جلسة (إلزامي)

1. commit لكل شغل سليم (لا تترك شغلاً معلقاً بلا commit).
2. علّم المهام المكتملة `[x]` في هذا الملف.
3. أضف سطراً في "سجل الجلسات" أسفل الملف.
4. إن اكتشفت مهمة ناقصة أثناء العمل: أضفها للمرحلة المناسبة بترقيم
   فرعي (مثل T2.3) — لا تنفّذها فوراً إلا إن كانت blocker.

## قواعد العمل الإلزامية

1. **مهمة واحدة في كل مرة.** لا استثناءات.
2. **commit بعد كل مهمة ناجحة**: `feat(plan): T<رقم> — <وصف>`
3. **لا تلمس ما لم يُطلب**: routing، API، صفحات خارج المهمة الحالية.
4. **معيار القبول شرط للتعليم `[x]`**: تحقق فعلياً (متصفح/curl) لا
   نظرياً.
5. **الفشل مرتين = BLOCKED**: اكتب `BLOCKED: <السبب>` تحت المهمة
   وانتقل لمهمة مستقلة. لا تدور بحلقة محاولات.
6. **الحَكَم البصري**: ملفات `handoff-designs/*.html`. عند أي تعارض
   بين وصف نصي والـHTML المرجعي، الـHTML يفوز.
7. **الحَكَم المنطقي**: `CLAUDE.md` (الألوان، القوائم، RFQ-per-View،
   قاعدة الأرقام المالية، نمط الموبايل المزدوج).
8. **RTL دائماً**: `text-start/end`، `ps-*/pe-*`، `ms-*/me-*` بدل
   left/right الثابتة.
