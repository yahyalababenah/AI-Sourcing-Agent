# Prompt لـClaude Code — انسخه كما هو في بداية كل جلسة

اقرأ الملفين التاليين بالترتيب قبل أي شيء:
1. `PLAN.md` — خطة العمل الكاملة وحالة التقدم
2. `CLAUDE.md` — قرارات التصميم والقواعد الثابتة

ثم نفّذ "بروتوكول بداية الجلسة" الموجود أعلى PLAN.md:
- افحص git log وgit status
- تأكد أن npm run dev يعمل
- ابدأ من أول مهمة [ ] غير مكتملة

قواعد غير قابلة للكسر:
- مهمة واحدة فقط في كل مرة، commit بعد كل مهمة ناجحة
- لا تعلّم [x] قبل التحقق الفعلي من معيار القبول
- عند إنهاء أي مهمة: حدّث PLAN.md (علّم [x] + سجل الجلسات)
- الفشل مرتين = اكتب BLOCKED وانتقل لمهمة مستقلة
- المرجع البصري النهائي: handoff-designs/*.html

عندما تكمل كل مهام جلستك أو يمتلئ السياق: توقف، حدّث سجل الجلسات
في PLAN.md، واعمل commit أخير. الجلسة القادمة ستكمل من حيث توقفت.
